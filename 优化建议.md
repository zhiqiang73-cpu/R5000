### 🔥 优先级2：动态移动止盈

**问题**：
- 当前止盈是固定的，赚了就跑
- 第2笔虽然赚了123.46，但可能本来可以赚更多

**方案**：
```python
# 当盈利 > 1.0 ATR 时，启动移动止损
# 移动止损 = 当前价格 - 0.5 ATR
# 让利润奔跑，同时保护利润

def check_trailing_stop(self, current_price, entry_price, atr_1min):
    """检查是否触发移动止损"""
    profit = abs(current_price - entry_price)

    # 盈利超过1倍ATR时启动移动止损
    if profit > atr_1min:
        self.trailing_stop_active = True
        self.trailing_stop_price = current_price - atr_1min * 0.5

    # 检查是否跌破移动止损
    if self.trailing_stop_active:
        if current_price <= self.trailing_stop_price:
            return True, "trailing_stop"

    return False, None
```

**预期影响**：
- 大赢的笔数会赚更多
- 小赢的笔数不会变差
