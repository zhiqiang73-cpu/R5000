## 🔍 当前问题诊断

### 1. PnL计算问题 ⚠️

**当前计算**（live_trader.py:_log_trade_pnl）：
```python
# 简单计算，未扣除手续费！
if side == "BUY":
    pnl = (exit_price - entry_price) * qty
else:  # SELL
    pnl = (entry_price - exit_price) * qty
```

**问题**：
- ❌ **没有扣除手续费**！
- ❌ 回测扣了手续费（0.15%），模拟盘没扣
- ❌ 导致模拟盘PnL虚高

**实际PnL应该是**：
```python
# 双边手续费：0.10%（Taker 0.05% * 2）
fee_rate = Decimal("0.001")  # 0.1%

if side == "BUY":
    gross_pnl = (exit_price - entry_price) * qty
    fee = entry_price * qty * fee_rate + exit_price * qty * fee_rate
    pnl = gross_pnl - fee
else:  # SELL
    gross_pnl = (entry_price - exit_price) * qty
    fee = entry_price * qty * fee_rate + exit_price * qty * fee_rate
    pnl = gross_pnl - fee
```

### 2. 止盈止损逻辑问题 ⚠️

**当前逻辑**（live_trader.py:_position_monitor）：
```python
# 固定止损止盈，没有动态调整
if signal.side == "BUY":
    if cur <= signal.stop_loss:
        close_reason = "软件止损"
        break
    if cur >= signal.take_profit:
        close_reason = "软件止盈"
        break
```

**问题**：
- ❌ **没有动态止盈**！
- ❌ 价格大幅盈利时，不会移动止盈保护利润
- ❌ 可能导致大赢变小赢

**例子**：
```
入场：70000
止盈：70500（+500）
止损：69800（-200）

场景1：价格涨到70800
  当前：止盈仍在70500
  结果：价格回调到70400，只赚500
  理想：应该移动止盈到70600，赚600

场景2：价格涨到71000
  当前：止盈仍在70500
  结果：价格回调，只赚500
  理想：应该移动止盈到70800，赚800
```

breakeven_stop = False

    current_price = get_current_price()

    if signal.side == "BUY":
        profit = current_price - signal.entry_price
    else:
        profit = signal.entry_price - current_price

    # 盈利超过1倍ATR，启动移动止盈
    if not trailing_stop_active and profit > atr_1min:
        trailing_stop_active = True
        trailing_stop_price = current_price - atr_1min * 0.5
        logger.info(f"启动移动止盈: {trailing_stop_price}")

    # 盈利超过2倍ATR，移动到保本
    if not breakeven_stop and profit > atr_1min * 2:
        breakeven_stop = True

        logger.info(f"移动止损到保本: {signal.stop_loss}")

    # 检查移动止盈
    if trailing_stop_active:
        if signal.side == "BUY" and current_price <= trailing_stop_price:
            close_position("trailing_stop")
            break
        elif signal.side == "SELL" and current_price >= trailing_stop_price:
            close_position("trailing_stop")
            break

**预期效果**：
