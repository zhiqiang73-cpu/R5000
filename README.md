## 🎯 系统概述

基于清算数据的 BTCUSDT 短线量化交易系统，采用三层决策架构：

```
Layer 1: 环境过滤 → Layer 2: 冲击分类 → Layer 3: 执行风控
```

**核心策略**：
- **均值回归**：无清算 = 过度反应，反向交易
- **趋势跟随**：有清算 = 真突破，顺势交易

## 📊 关键参数速查

### 时间止损（v2.0）

| 策略 | time_stop | 说明 |
|------|-----------|------|
| 均值回归 | 180秒 (3分) | 市价入场，快速回归 |
| 强清算趋势 | 900秒 (15分) | 市价追入 ✅ 已优化 |
| 普通趋势 | 1200秒 (20分) | 回调25% ✅ 已优化 |

### 信号分级

| 等级 | 风险 | 条件 |
|------|------|------|
| A | 1.5% | score >= 6 |
| B | 1.0% | score >= 4 |
| C | 0.5% | score >= 2 |

### 熔断规则

- 日亏损 > 3% → 停止当日交易
- 连续3笔亏损 → 暂停至次日0点
- 最近10笔胜率 < 30% → 暂停1小时
- 单笔亏损 > 2% → 暂停30分钟 + 人工确认
