### 第1步：实施动态杠杆（最有效！）

**时间**：30分钟
**代码**：
```python
# 在 src/risk/position.py 中
def calculate_position_dynamic_leverage(
    grade, stop_distance, account_balance, current_price, confidence=Decimal("0.5")
):
    # 根据信心度动态调整杠杆
    if confidence > Decimal("0.8"):
        leverage = 3
        risk_pct = Decimal("0.02")
    elif confidence > Decimal("0.6"):
        leverage = 2
        risk_pct = Decimal("0.015")
    else:
        leverage = 1
        risk_pct = Decimal("0.01")

    risk_usd = account_balance * risk_pct
    position = risk_usd / stop_distance
    max_position = account_balance * leverage * Decimal("0.8") / current_price
    return min(position, max_position)
```

**修改**：
1. `src/layers/executor.py` - 传递confidence
2. `src/risk/position.py` - 使用动态杠杆
