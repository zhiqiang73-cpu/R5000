# PnL计算修复完成报告

## 🎯 修复完成

### 问题诊断

**发现**：模拟盘PnL计算没有扣除手续费！
- 回测：扣除0.15%双边手续费
- 模拟盘：没有扣除 ❌
- 导致：模拟盘PnL虚高

### 修复内容

**文件**：`src/live_trader.py:_log_trade_pnl()`

**修改**：
```python
# 之前
pnl = (exit_price - entry_price) * qty  # 没扣手续费

# 现在
FEE_RATE = Decimal("0.001")  # 0.1%双边
fee = (entry_price * qty + exit_price * qty) * FEE_RATE
pnl = gross_pnl - fee  # 真实PnL
```

## 📊 影响评估

